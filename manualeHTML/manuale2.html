Introduzione ad HTML

HTML è l'acronimo di HyperText Markup Language e, come si intende dall'acronimo, viene considerato un linguaggio di markup e non di programmazione. L'obbiettivo di questo linguaggio dunque non è quello di costruire programmi che risolvano un determinato problema, il fine di HTML è invece quello di indicare la posizione di elementi all'interno di una pagina attraverso appositi marcatori, detti TAGs. Questi elementi devono essere inclusi tra parentesi angolari (ad esempio, < img > è il tag per un'immagine).

Struttura Base di una pagina HTML

Per strutturare una pagina HTML, come nella maggior parte dei linguaggi, si necessita di una struttura base per definire i tag principali e inserire elementi nella pagina.

Questa porzione di codice, una volta eseguita, restituirà una semplice pagina bianca priva di elementi. Per "riempire" la pagina si necessita dei tag, quelli principali sono:

TAGs Utilizzo
< html > Questo tag racchiude l'intero documento HTML, e principalmente comprende il document header, rappresentato dai tags < head >...< /head > ed il document body, rappresentato dai tags < body ...< /body >.
< head > Indica il document header, può contenere all'interno altri tags HTML come < title >, < meta > ecc.
< link > Il tag < link > viene usato nelle pagine per creare un collegamento tra il file HTML e lo stylesheet CSS (cascading style sheet). Attraverso il file CSS è possibile personalizzare la pagina, aggiungendo elementi decorativi oppure colorando alcune parti del documento. E' possibile utilizzare CSS anche nello stesso file HTML, attraverso il tag < style >, all'interno di questo tag la sintassi sarà la stessa del CSS.
< html > Il tag < title > è utilizzato all'interno del tag < head > per indicare il titolo della scheda
< body > Rappresenta il corpo del documento HTML e contiene altri tags come < h1 >, < div >, < p >
< h1 ... 6> Da h1 a h6 sono tag per impostare il titolo del documento. Sono utilizzati solitamente in ordine crescente: h1 crea un titolo molto grande, di solito quello principale della pagina, h6 crea il titolo più piccolo, utilizzato solitamente per scrivere sottotitoli nei paragrafi.
< p > Il tag p viene utilizzato per scrivere dei paragrafi nella pagine. Tra due tag < p > si formerà uno spazio di alcuni px per separare i due paragrafi, questo tag può dunque essere usato anche semplicemente per distanziare elementi quali tabelle, elenchi, immagini, ecc.
< img > Questo tag viene usato per inserire immagini nella pagina HTML, la sorgente può essere sia un file nel computer, sia il link di un'immagine trovata sul web.
< a > Il tag < a > (anchor) viene usato nelle pagine per creare un collegamento ipertestuale nel documento, o anche esterno. Può essere usato per linkare immagini sul pc, immagini online, link di siti o paragrafi nello stesso documetno attraverso l'attributo id

Chiaramente nel linguaggio HTML sono presenti migliaia di marcatori che permettono di fare altrettante cose e inserire altrettanti elementi. Ad esempio, si possono fare elenchi puntati con il tag < ul > < li >...< /li > < ul >, oppure tabelle con il tag < table > < tr > < td >...< /td > < /tr > < /table >. Ce ne sono anche di molto particolari come il tag < marquee > che crea un testo scorrevole nella pagina HTML.

Gli Attributi ai TAGs

Per rendere ancora più personalizzabili e vari gli elementi e i marcatori nella propria pagina, sono disponibili i cosiddetti attributi. Un'attributo viene collocato nel tag di apertura ed è composto da un nome e un valore.

Esempio di attributi al tag < table >.
L'attributo id serve per identificare univocamente la tabella,
l'attributo border serve per assegnare un bordo alla tabella,
l'attributo cellspacing serve per definire lo spazio tra le celle,
mentre l'attributo align serve per allineare la tabella, in questo caso al centro.

Il nome indica la propietà da impostare all'elemento (align), mentre il valore indica il valore da attribuire alla propietà (center).
